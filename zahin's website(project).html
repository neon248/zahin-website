Projects

Face Recognition System using MATLAB

Used Histogram Oriented Gradient (HOG) features to train a Matlab program to recognize a person from an image. A dataset consisting images of 7 different people was used to train the system. This enabled the system to recognize an individual within that group.

Password Protection System Using Microcontroller 8051

Designed a password protection system in Proteus software and then implemented it using Microcontroller 8051. The user had to provide a correct combination of ID and password through a keypad to get access from the Microcontroller.

Calculator Using Digital Logic Gates on a PCB

Designed a calculator capable of simple addition, subtraction, multiplication and division of two digit numbers using logic gates. Proteus simulation software was used to design the calculator and implementation was done on a PCB using IC logic gates.

Hand Gesture Recognition System

A hand gesture recognition system which converted the hand gestures on video into text and audio. The numbers from one to five were set as classifications. What number the subject is showing with their hand would be detected and identified by the system.
